## 任务
为每个平台生成一个 JSON 对象，最终输出一个 JSON 数组。每个对象字段：
- platform: 平台标识，必须是 "xiaohongshu"、"toutiao" 或 "wechat" 之一
- title: 平台适配标题
- body: 平台适配正文
- tags: 标签数组，4-6 个
- image_prompt: 配图提示词（可选）

各平台要求：
1. 小红书：350-600 字，口语化、真实感、emoji、分段清晰，结尾带 #话题
2. 今日头条：400-700 字，【今日观察】开头，信息密度高，引用多方事实，结尾引导讨论
3. 微信公众号：1000-1800 字 Markdown，frontmatter + 热点背景 + 核心看点 + 深度解读 + 读者价值

通用要求：
- 综合多篇资料的关键信息，进行交叉验证和整合，不要简单堆砌标题
- 提炼出单一资料看不到的视角和判断
- 体现用户的创作意图

只输出 JSON 数组，不要解释。
