## 使用规则

1. 新增平台、数据源、生成器、发布命令或外部依赖时，必须更新相关文档。
2. 任何真实平台动作都必须先在文档中说明风险、输入、输出、审批门禁和回滚方式。
3. 文档不得记录账号 Cookie、验证码、AppSecret、Access Token 或真实用户隐私数据。
4. 每次阶段性交付后，应补充验收证据，例如测试命令、截图路径、接口响应摘要和剩余风险。

## 当前安全模式

当前系统保持 preview-only 或 draft-only：

- 后端适配器只生成预览载荷。
- 前端只展示预览、状态和日志。
- 外部平台工具暂不由系统自动触发真实发布。
- 真实发布需要单独开发审核工作流和人工确认门禁。
