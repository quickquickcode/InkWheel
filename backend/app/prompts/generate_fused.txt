## 任务
为每个平台生成一个 JSON 对象，最终输出一个 JSON 数组。每个对象字段：
- platform: 平台标识，必须是 "xiaohongshu"、"toutiao" 或 "wechat" 之一
- title: 平台适配标题
- body: 平台适配正文
- tags: 标签数组，3-5 个
- image_prompt: 配图提示词（可选）

要求：
1. 综合多篇资料的关键信息，不要简单堆砌。
2. 体现用户的创作意图。
3. 小红书版本要口语化、有种草感、带 emoji 风格。
4. 今日头条版本要新闻快评风格，信息密度高。
5. 微信公众号版本要正式、有结构、适合长文阅读。

只输出 JSON 数组，不要解释。
